# Cliente Oficial Retro76

OTClientV8 para **retro76.cl:7171** (Tibia 7.6).

## Descargar

| Plataforma | Carpeta | Lanzador |
|------------|---------|----------|
| macOS | `mac/` | `Iniciar Cliente.command` (requiere XQuartz) |
| Windows | `windows/` | `Iniciar Cliente.bat` |

## Configuración

- Servidor: **Cliente Oficial Retro** → `retro76.cl:7171:760`
- Fondo personalizado Retro76
- Sin selector de otros servidores
- Barras de vida/mana estándar (panel derecho + opciones por defecto OTC)
- Audio desactivado por defecto

## macOS

1. Instalar [XQuartz](https://www.xquartz.org/)
2. Descomprimir `mac/`
3. Doble clic en `Iniciar Cliente.command`

## Windows

1. Descomprimir `windows/`
2. Doble clic en `Iniciar Cliente.bat`
